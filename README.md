# 📝 To-Do List App (Python & Tkinter)

A simple **To-Do List Application** built using Python's Tkinter for a GUI-based task management experience.

## 🚀 Features
✅ Add new tasks  
✅ Remove selected tasks  
✅ Clear all tasks  
✅ Save tasks to a file  
✅ Load tasks from a file
